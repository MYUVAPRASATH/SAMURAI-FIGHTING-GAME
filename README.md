Samurai Fighting Game
Overview
Welcome to Samurai Fighting Game! This project aims to provide an action-packed gaming experience where players can immerse themselves in the world of master samurai warriors. With intuitive controls and engaging gameplay, this game is designed to entertain both kids and adults alike.

Features
Action-packed gameplay: Take control of a master samurai warrior and engage in one-on-one combat with your opponent.
Intuitive controls: Use simple keyboard controls to move your character, perform attacks, and defend against your opponent's moves.
Special abilities: Gain an edge in battle by utilizing special abilities unique to each character.
Two-player mode: Challenge your friends in local multiplayer mode and see who emerges victorious in the ultimate showdown.
How to Play
Getting Started: Run the project to access the game's front page.
Game Page: Click the "Start" button to enter the game page.
Controls:
Player 1:
Jump: W
Move Left: A
Move Right: D
Attack: SPACE
Player 2:
Jump: Up Arrow
Move Left: Left Arrow
Move Right: Right Arrow
Attack: 0
